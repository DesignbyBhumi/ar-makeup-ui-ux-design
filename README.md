AR Virtual Try-On (UI/UX Project)

 Overview
This project focuses on designing a user-friendly AR-based virtual try-on experience for users to try products digitally.

 Tools Used
- Figma
- Prototyping

 Features
- Product selection
- AR try-on interface
- User-friendly navigation
- Clean UI design
